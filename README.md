  * Языки: Python, SQL
  * Анализ данных: библиотеки Pandas, NumPy, SciPy, Statsmodels

  * А/В-тестирование
  * Работа с гипотезами
  * Машинное обучение: библиотеки Scikit-learn, XGBoost
